## :boom: Start

- ### **Requirements**

  - Install **[Git](https://git-scm.com/)**
  - Install **[NodeJS](https://nodejs.org/en/)**

**1. Clone this repository:**

```sh
  $ git clone https://github.com/frassaolucas/entrevista-fe.git
```

**2. Intall dependencies**

```sh
  $ npm install
```
ou
```sh
  $ yarn
```

**3. Run project locally**

```sh
  $ npm run start
```
ou
```sh
  $ yarn start
```
